# TasksTracker

TasksTracker is a .NET MAUI app for recording tasks with descriptions, start times, and durations. It stores entries locally in LiteDB and provides tools to search, filter, export, and back up your data.

## Features

- Create tasks with description, start time, and duration.
- Voice-to-text input for task descriptions.
- View all tasks on a dedicated page with search and date filtering.
- Export tasks to Excel with monthly hour summaries (German column labels).
- Backup and import the local database file.

## Tech Stack

- .NET MAUI
- LiteDB
- CommunityToolkit.Maui
- ClosedXML
